# 📊 Adventure Works Sales Dashboard

## Project Overview
This Power BI dashboard provides a comprehensive analysis of Adventure Works sales performance, profitability, and product trends.

## Dashboard Preview

![Adventure Works Dashboard](Screenshot%20(562).png)

## Key Metrics
- Total Sales: $29.36M
- Total Orders: 60K
- Total Profit: $12.08M

## Features
- Sales by Category
- Sales by Subcategory
- Profit Analysis
- Top 10 Products Analysis
- Interactive Filters (Year, Month, Territory)
- Dynamic Power BI Visualizations

## Tools Used
- Power BI
- Power Query
- DAX
- Data Modeling
